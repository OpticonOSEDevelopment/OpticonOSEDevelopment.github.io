OPN-2500 and OPN-6000 Development Kit

This comprehensive C development kit is tailored for Opticon's OPN-2500 and OPN-6000 barcode scanners. It offers a suite of tools and examples to facilitate efficient application development, including guidance on installation, application building, barcode reading, and communication protocols.

OptiConnect SDK for Android

The OptiConnect SDK enables seamless integration with Opticon's OPN-2500 and OPN-6000 BLE barcode scanners on Android platforms. It provides functionalities for Bluetooth discovery, real-time barcode data streaming, and programmatic control of scanner settings, supporting development in both Kotlin and Java.

  • GitHub Repository
  • Manual

OptiConnect SDK for Flutter

Designed for Flutter developers, the OptiConnect SDK facilitates integration with Opticon's OPN-2500 and OPN-6000 BLE barcode scanners. It offers features such as Bluetooth discovery, real-time data streaming, and command control, streamlining the development process for cross-platform applications.

OPN Companion SDK for Windows, OS X & Web applications

This SDK from for the Companion Scanners makes it possible to create and customize Windows, OS X and Web applications. It offers a suite of tools and examples to create applications to connect to, extract barcode data and configure your devices.
